## üìã Project Overview
A Laravel-based web application that translates English text to multiple languages and converts the translated text to speech, with audio playback in the browser.

---

## üèóÔ∏è Technology Stack

### Backend
- **Framework**: Laravel 10.x
- **Database**: MySQL
- **PHP**: 8.1+
- **APIs**: 
  - Translation: Google Translate API (via Stichoza/google-translate-php - FREE, no API key needed)
  - Text-to-Speech: Google Cloud Text-to-Speech API (Free tier: 1M characters/month)
  - Alternative TTS: VoiceRSS API (Free tier: 350 requests/day) or Web Speech API (browser-based, completely free)

### Frontend
- **CSS Framework**: Tailwind CSS
- **JavaScript**: Alpine.js for interactivity
- **Audio Player**: HTML5 Audio API
- **Icons**: Heroicons or Font Awesome

### Deployment
- **Platform**: Railway / Heroku / DigitalOcean App Platform (all have free tiers)
- **Alternative**: Shared hosting with Laravel support

## üîÑ Application Flow

1. **User Input** ‚Üí User enters English text and selects target language(s)
2. **Validation** ‚Üí Laravel validates input (required, max length, language selection)
3. **Translation** ‚Üí TranslationService translates text to each selected language
4. **Text-to-Speech** ‚Üí TextToSpeechService converts translated text to audio
5. **Storage** ‚Üí Audio files saved to storage/app/public/audio
6. **Database** ‚Üí Translation record saved to MySQL
7. **Response** ‚Üí Return translated text + audio file URLs to frontend
8. **Display** ‚Üí Show results with audio players and download buttons
9. **History** ‚Üí Save to history for later retrieval
